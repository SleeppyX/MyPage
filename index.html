ชื่อ-นามสกุล : นายสวรุจ สองเมือง

สาขาวิชา : วิศวะคอมพิวเตอร์และปัญญาประดิษฐ์

กลุ่มเรียน : G1

เหตุผลที่อยากเรียน Full stack web development :

- อยากเพิ่มความรู้และฝึกทักษะการเขียนโปรแกรม เพื่อนำไปใช้ทำงานเกี่ยวกับ web และเพิ่มทักษะในการเขียนภาษาต่างๆ

ความคาดหวังต่อวิชานี้ :

- จะได้มีความรู้ในการเขียนโปรแกรมมากขึ้น ทำให้เขียน code ได้หลายภาษา นำไปใช้ในการทำงานในอนาคต

อะไรจุดอ่อน ที่ตนเองต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา :

- การไม่เข้าใจคำศัพท์ต่างๆหรือวิธีการคิดในภาษา code ทำให้ตามไม่ทัน แนวทางการพัฒนา หมั่นศึกษาหาความรู้เรื่องคำศัพท์ต่างๆเพิ่มมากขึ้น
